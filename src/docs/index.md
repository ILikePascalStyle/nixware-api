# Home

Nixware Lua API documentation.

---

# Loading Lua scripts

1. Drop script into `C:\nixware\scripts` folder.
2. Click `refresh` in actions list on cheat menu.
3. Select the script from list and press `load script`.

---

# LuaJIT
